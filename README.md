## What Is StreamLens?

StreamLens is a **Netflix-grade two-stage search and recommendation system** built from scratch вҖ” covering the full ML lifecycle from raw interaction data through curated training pairs, gated model promotion, real-time serving, multilingual GenAI explanations, and diffusion model poster generation.

**Data flow:** `ingest вҶ’ store вҶ’ retrieve вҶ’ rerank вҶ’ infer вҶ’ feedback`

**Headline numbers:**
- LTR nDCG@10 = **0.9300** вҖ” exceeded target of 0.80 by 16.3%
- **21 ML algorithms** вҖ” retrieval, ranking, personalisation, causal inference, visual AI, generative AI
- **106 API endpoints** вҖ” search, explanation, feed, VLM, SQL explorer, diffusion, causal, self-healing
- **44 languages** вҖ” GPT-4o-mini explanations in pure target script, zero mixing
- **RAGAS**: Faithfulness=0.705 В· Relevance=0.752 В· Recall=1.000 вҖ” all targets met
- **Diffusion pipeline** вҖ” DDPM noise schedule (pure numpy) + DALL-E 3 HD 1024Г—1792
- **Multi-modal AI** вҖ” CLIP + GPT-4o vision + OpenAI TTS + Whisper + DALL-E 3
- **Self-supervised learning** вҖ” contrastive fine-tuning of e5-base-v2 (+18.4% dense nDCG)
- **Data curation engine** вҖ” PySpark 33.8M вҶ’ 1.29M co-watch pairs, 9 quality gates
- **SQL Explorer** вҖ” live at `/sql`, 8 production tables, 10 real queries

### DDPM Noise Schedule (Ho et al. 2020 вҖ” pure numpy)

```python
# diffusion_pipeline.py
betas = np.linspace(0.0001, 0.02, 1000)          # linear schedule
alphas_cumprod = np.cumprod(1.0 - betas)          # бҫұ_t

# Forward: q(x_t|x_0) = N(x_t; вҲҡбҫұ_tВ·x_0, (1-бҫұ_t)В·I)
# Reverse: p_Оё(x_{t-1}|x_t) = N(Ој_Оё(x_t,t), ОЈ_Оё(x_t,t))
# Inference: x_T ~ N(0,I) вҶ’ UNet predicts Оө_Оё вҶ’ denoise TвҶ’0 вҶ’ x_0
```

## Postmortem вҖ” What Broke and How I Fixed It

### Incident 1 вҖ” OpenAI Key Returning 429 / Garbage Explanations

**What broke:** Explanations returned broken template: *"If you love likes feel-good romance, Adventure film known for pixar will be your kind of night."*

**Root cause:** Two `OPENAI_API_KEY` entries in `.env`. Docker read the first (expired). Format string `{pref}` didn't match variable name.

**Fix:** Dedup script for `.env`, explicit key injection at startup, exponential backoff retry (1.5sвҶ’3sвҶ’6sвҶ’12s), Redis 7-day cache, fixed fallback template.

### Incident 2 вҖ” TMDB Posters All Black (401)

**Root cause:** `git filter-repo` replaced TMDB key in `index.html` with literal string `TMDB_API_KEY_REMOVED`.

**Fix:** `sed -i '' "s/TMDB_API_KEY_REMOVED/real_key/" src/app/demo_ui/index.html`. Lesson: grep all source files after `git filter-repo`.

### Incident 3 вҖ” Docker Not Picking Up New `.env`

**Root cause:** Docker Compose cached resolved environment. Old key baked into image layer.

**Fix:** `OPENAI_API_KEY=<key> docker compose up -d` вҖ” shell variable overrides Docker `.env` cache. Added to `start.sh`.

### Incident 4 вҖ” RAGAS Scores All 0.000

**Root cause:** Script called `/answer` which requires Ollama. Word-overlap scoring underestimates GPT paraphrases.

**Fix:** Switched to `/search` + `/explain` (GPT-4o-mini). Replaced word overlap with semantic cosine via `all-MiniLM-L6-v2`.

**Result:** F=0.705, R=0.752, C=1.000 вҖ” all targets met.

### Explain your diffusion model work

DDPM linear beta schedule: ОІ from 0.0001 вҶ’ 0.02 over T=1000 steps. At t=0: SNR=9999 (clean). At t=999: SNR=0.00004 (pure noise). Inference: start from x_T~N(0,I), run UNet TвҶ’0 to predict and subtract noise. DALL-E 3 uses same DDPM principle in 64Г—64 latent space (VAE encode вҶ’ diffusion вҶ’ VAE decode вҶ’ HD image). Genre-aware prompt engineering maps Crime вҶ’ neo-noir, Sci-Fi вҶ’ cyberpunk. No text in prompt вҖ” DALL-E 3 can't render text reliably, UI overlays title instead.
