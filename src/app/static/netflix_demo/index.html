What you’re looking at

1) Hybrid Retrieval

Combines BM25 lexical matching with dense semantic embeddings. This makes search robust to typos + natural language (“feel good family movie”).

2) LambdaRank (LTR)

Re-ranks the top candidates using learned features to optimize nDCG@10. You show the lift in the top bar (offline evaluation).

3) Grounded + Agentic RAG

“Why this?” answers are generated ONLY from retrieved metadata (citations). If grounding is weak, the agent retries with more candidates/context.

Multilingual Voice

Browser voices often fail for Tamil/Telugu/Malayalam. This demo uses server-side edge-tts for consistent multilingual TTS.
