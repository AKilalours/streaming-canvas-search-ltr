Forward & Reverse Process
-- Forward (training): add noise q(x_t | x_0) = N(x_t; √ᾱ_t · x_0, (1-ᾱ_t)·I) -- Reverse (inference): denoise p_θ(x_{t-1} | x_t) = N(μ_θ(x_t,t), Σ_θ(x_t,t)) -- β linear: 0.0001 → 0.02 over T=1000 ᾱ_t = ∏ αᵢ, αᵢ = 1 - βᵢ
Inference Algorithm
①
Sample x_T ~ N(0, I) — start from pure Gaussian noise
②
For t = T, T-1, ..., 1: run UNet(x_t, t, text_emb) → predict noise ε_θ
③
Compute: μ = (x_t - β_t/√(1-ᾱ_t)·ε_θ) / √α_t
④
Sample: x_{t-1} = μ + √β_t · z where z ~ N(0,I)
⑤
Return x_0 — the generated image ✅
